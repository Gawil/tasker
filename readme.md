# tasker

Apache server must have read and write permissions on database folder AND every file it contains.
The database folder must be like this :

./database/ :
  ./database/dead/
  ./database/done/
  ./database/lang/ :
    ./database/lang/en.php
    ./database/lang/fr.php
  ./database/todo/
  ./database/users/
  ./database/wip/
  
  ./database/config
  ./database/email
  ./databse/passwd
